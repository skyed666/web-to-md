## 📝 详细描述（Description，必填，≤16,000 字符）
```
网页转换器让你一键把任意网页变成 Markdown 或 JSON，专为喂给 ChatGPT、Claude 等 AI 大模型设计。

✨ 核心功能
• 网页 → Markdown：提取干净正文，去掉广告、导航等干扰
• 网页 → JSON：返回结构化数据 {title, url, content, ...}，便于程序处理
• 一键复制到剪贴板
• 一键下载为 .md / .json 文件
• 自动填入当前标签页网址，也可手动粘贴任意 URL

🛡️ 智能混合抓取
• 优先使用云端服务提取（输出最干净）
• 当目标网站开了反爬验证（如 Cloudflare），自动切换为"本地模式"——由你的浏览器直接抓取页面并本地转换，绕过反爬限制
• 转换完成后会用颜色标记本次来源：绿色=云端，琥珀色=本地

🔒 隐私优先
• 纯前端，无后端服务器，不收集你的任何个人数据
• 不读取浏览历史、不存储账号信息
• 所有请求均通过 HTTPS 加密
• 详细隐私政策见扩展介绍页

🎯 适用场景
• 把网页文章喂给 ChatGPT / Claude 等做总结、翻译、改写
• 收集资料时把网页存成 Markdown 归档
• 数据提取时把网页结构化成 JSON
• 绕过部分网站的反爬限制获取正文

📌 使用方法
1. 打开任意网页
2. 点击工具栏的扩展图标
3. 点"转 Markdown"或"转 JSON"
4. 预览结果，点"复制"或"下载"

💡 说明
• 云端模式基于 Jina AI Reader API，免费额度 20 次/分钟（按 IP）
• 本地模式无限速，但输出质量略低于云端（不渲染 JavaScript 动态内容）
• 浏览器内部页（chrome:// 等）无法转换，请转换普通网页

兼容 Google Chrome 与 Microsoft Edge。
```

## 📝 隐私政策 URL（必填）
你需要把 `PRIVACY_POLICY.html` 上传到一个可公开访问的地方，然后填这个 URL。推荐方式：
- **方式 1（推荐）**：上传到你的 GitHub 仓库，用 GitHub Pages 开启后填 `https://你的用户名.github.io/仓库名/PRIVACY_POLICY.html`
- **方式 2**：复制内容到你自己的网站/博客
- **方式 3**：用免费静态托管（如 Netlify、Vercel、腾讯云 COS 静态网站）上传
